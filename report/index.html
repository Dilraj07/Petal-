Pipeline

What Petal Did

01

IR Generation

Initialized LLVM 22.0 frontend and generated ClangIR from the target C source file.

02

Hardware Telemetry

Profiled against AMD uProf metrics. Baseline PPT: 45.2W. Thermal spikes from cache thrashing.

03

ML Prediction

Offloaded to Ryzen AI NPU. XGBoost identified inefficient loop nesting, predicting 38% savings.

04

IR Rewrite

Applied PetalEnergyOptimizationPass — swapped naive O(N³) loop with blocked, cache-friendly iteration.
