Pipeline

What Petal Did

01

Hotspot Detection

Analyzes C source for O(N³) nested loops and cache-inefficient patterns using regex-based static analysis.

02

Policy Decision

Eco, Balanced, or Performance profiles determine optimization aggressiveness based on confidence thresholds.

03

Loop Tiling

Applies cache-blocking transformations with configurable tile sizes (32-128) for improved data locality.

04

Hardware Telemetry

Profiles real energy consumption via AMD uProf, Intel RAPL, Apple Silicon, or synthetic CPU modeling.
